Installation¶

Currently, to install the package, the github repository must to be cloned. It is recommended that this be done into an empty directory with a new virtual enviornment. When in the new directory, type:

git clone https://github.com/mcnichon/edgefinder.git

More detailed instructions pertaining to cloning github repositiroies can be found ‘here’_ .._ here: https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository
